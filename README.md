# Architrons-AI-Prediction
Technical Details
Seluruh source code proyek Architrons tersedia di folder `/Architrons-Construction-AI`, yang meliputi:
- **Model Training:** Proses EDA dan pelatihan model Random Forest (`.ipynb`).
- **Web Application:** Implementasi model ke dalam aplikasi web berbasis Flask (`app.py`).
- **Dataset:** Data historis proyek konstruksi yang digunakan untuk pelatihan.
